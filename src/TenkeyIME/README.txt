TIME (Tenkey IME) 仕様書

前提
(*)キー配置は次のようなものとします。
|---|---|---|---|
|Num| / | * | - |
|---|---|---|---|
| 7 | 8 | 9 | + |
|---|---|---|   |
| 4 | 5 | 6 | + |
|---|---|---|---|
| 1 | 2 | 3 | E |
|---|---|---|   |
| 0   0 | . | E |
|---|---|---|---|
(*)NumLockキーは、入力モードを変更するのに使用します。

キー配置
NumLockオン時
(*)第一打鍵時(子音のキー配置)
|---|---|---|---|
|   |   |   |   |
|---|---|---|---|
| k | s | t |   |
|---|---|---|   |
| n | x | h |   |
|---|---|---|---|
| m | y | r |   |
|---|---|---|   |
| w   w | . |   |
|---|---|---|---|
但し、xは無子音(あ、い、う、え、お)を表し、.は記号を入力する際に使うことを表します。
また、子音の入力に関係ない場合はスペースを使用しています。
(*)第二打鍵時(母音のキー配置)
|---|---|---|---|
|   |   |   |   |
|---|---|---|---|
|(!)| u |(@)|   |
|---|---|---|   |
| i | a | e |   |
|---|---|---|---|
|   | o |   |   |
|---|---|---|   |
|       |   |   |
|---|---|---|---|
濁点付きの場合は、ひらがな入力後「(@)」を入力します。半濁点にする場合はもう一度入力します。「ん」を入力する場合はnの後直接「(@)」を入力します。
小文字を入力するには、ひらがな入力後、「(!)」を入力します。
(*)第二打鍵時(.打鍵時、記号のキー配置)

特殊キーについて
(*)-キーはバックスペースになります。
(*)Enterキーはエンターキーのままになります。
(*)NumLockオン時は/キーはカーソル左に、*キーはカーソル右になります。。NumLockオフ時は/キーはCapsLockに、*キーはTabになります。
(*)+キーは、文字未入力時はスペースに、文字入力時は変換キーになります。
